# ☕ Cafe Management System

A web-based Cafe Management System that helps automate cafe operations such as menu management, customer orders, billing, and inventory management.

## 📌 Features
- User Login & Authentication
- Menu Management
- Customer Order Management
- Billing System
- Inventory Management
- Order History
- Responsive User Interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## 🚀 Installation
1. Download or clone this repository.
2. Move the project folder to the `htdocs` folder in XAMPP.
3. Start Apache and MySQL.
4. Import the SQL database into phpMyAdmin.
5. Open `http://localhost/Cafe-Management-System` in your browser.

To simplify and automate the day-to-day operations of a cafe by reducing manual work and improving efficiency.
